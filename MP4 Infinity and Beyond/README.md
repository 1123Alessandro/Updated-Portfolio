# CS 2608 Machine Project 4: Infinity and Beyond

A shopping page for watches. A project for showcasing the use of java servlets.

## Major Contributions

Contributed to a lot of the back-end features of the web-app. Including
1. JSP Scriptlets for view control
1. the `controller` package in source packages
1. MVC implementation throughout the entire web-app

## Setup

Use NetBeans and download the GlassFish Server within the IDE. Set the configuration of this project to run on the server using Java EE 5.

Clean and build, then run.
